# Week 10: JavaScript Logic Lab

## Project Description
This is a simple JavaScript application that interacts with the user to:
1. Provide a time-based greeting (Morning/Afternoon/Evening).
2. Calculate the approximate number of days the user has lived based on their birth year.
3. Determine if the user is an adult or a minor.

## How to Run
1. Clone this repository.
2. Open `index.html` using **Live Server** in VS Code.
3. Check the **Browser Console** (F12) to see the output.
